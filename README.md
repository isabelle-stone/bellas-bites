# Bella's Bites - React Restaurant Website

A responsive restaurant website built with React, showcasing modern web development practices and component-based architecture.

## Live Demos
**Original HTML Version:** https://bellasbites.netlify.app/  
**React Version:** [https://bellasbites-react.netlify.app/](https://bellas-bites.vercel.app/)

## Technologies
- React 18 with Hooks
- React Router for navigation  
- Vite build tool
- Custom CSS
- FontAwesome icons

## Features
- Interactive image slider with auto-play
- Shopping cart with add/remove functionality
- Contact form with validation
- Responsive hamburger menu
- Multi-page navigation

This project demonstrates full-stack development skills including React state management, responsive design, and modern deployment practices.
